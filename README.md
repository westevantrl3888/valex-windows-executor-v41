## Overview

Valex is a Windows-native Roblox script executor built around Lua injection, fast script launching, and a minimal workflow for users who prefer managing scripts from a desktop app. Its layout stays intentionally compact, with an easy startup process and a bundled script hub that provides a broad set of ready-to-run entries.

Rather than leaning on heavy visuals, the project emphasizes usable execution tools. With auto-update support, persistent queue storage, and a multilingual interface, Valex is aimed at keeping script handling tidy and efficient on Windows 10 and Windows 11.

## FAQ

**Is Valex limited to Windows?**  
Yes. The project is framed as a Windows desktop tool for Windows 10 and Windows 11.

**Does it update itself automatically?**  
Valex includes an auto-update engine intended to help the app stay aligned with newer builds.

**What sets it apart from other Roblox executors?**  
The main emphasis is on a lightweight interface, a large built-in script hub, persistent queue storage, and batch execution support.

**Can this put my Roblox account at risk?**  
Third-party script executors can carry account risk depending on how they are used. Check applicable rules and use your own judgment.

**Are scripts stored locally?**  
Yes. The persistent queue uses SQLite-backed storage, which allows queued items to remain available across sessions on the same machine.
